# env-core

EnvKit utility to get env variables for multiple sources

## Example Installation (NodeJS)

```bash
npm install @luolapeikko/env-core @luolapeikko/env-core-dotenv @luolapeikko/env-core-nodejs
```

## Example Installation (Browser)

```bash
npm install @luolapeikko/env-core @luolapeikko/env-core-vite @luolapeikko/env-core-fetch
```
